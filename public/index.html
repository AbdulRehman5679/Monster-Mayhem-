Game Rules

1. Each player takes turns moving a monster (V for Vampire, W for Werewolf, G for Ghost).

2. A player can move a monster to any unoccupied cell on the grid according to the rules.

3. Monsters can move any number of squares horizontally or vertically, or up to two squares diagonally.

4. Conflicts between monsters are resolved as follows:

  • If a Vampire (V) and a Werewolf (W) meet, the Werewolf is removed.
  • If a Werewolf (W) and a Ghost (G) meet, the Ghost is removed.
  • If a Ghost (G) and a Vampire (V) meet, the Vampire is removed.
  • If two monsters of the same type meet, both are removed.

5. A player is eliminated once all their monsters are removed.

6. The game ends when all other players are eliminated.
